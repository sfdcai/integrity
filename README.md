# DeepGuard Well Integrity Platform

DeepGuard is a production-ready SaaS blueprint for well integrity management built with FastAPI, SQLite, React + Vite, Tailwind, and D3.js.

## Features
- Well registry, annulus management, MAASP engine, barrier envelope tracking.
- JWT auth, reminders, live trend endpoints, D3 interactive schematic.
- Modern UI with dashboard, annulus cards, tasks, and charts.
- Ubuntu-native installer with PM2 process supervision.

See `docs/SETUP.md` and `docs/ARCHITECTURE.md` for full details.
